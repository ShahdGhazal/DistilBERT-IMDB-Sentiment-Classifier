# DistilBERT IMDb Sentiment Classifier

This project fine-tunes DistilBERT for binary sentiment classification on IMDb movie reviews.

## Goal

Classify movie reviews as:

- Positive
- Negative

## Tools Used

- Python
- PyTorch
- Hugging Face Transformers
- Hugging Face Datasets
- Google Colab

## Model

The project uses `distilbert-base-uncased`, a smaller and faster version of BERT.

## Dataset

IMDb movie review dataset.

## Status

Project in progress.
